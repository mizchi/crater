# Vendored test fonts

`minimal-kern.ttf` is a minimal TrueType test font vendored from
[`mizchi/font`](https://mooncakes.io/docs/mizchi/font) (Apache-2.0), used
only by the image-VRT runner to exercise real-font text rendering
deterministically in CI. It contains a small set of glyphs (e.g. A, V).
