# pkspec layout

This directory holds crater's pkspec authoring layer.

- `crater.pkl` declares Goals and Scenarios for local gates, browser compatibility, VRT, benchmarks, and release safety.
- `tasks.Test.pkl` imports the actual `tasks/*.pkl` pkfire task definitions and checks their graph metadata with `pkf graph --json`.
- `../pkspec/*.pkl` are vendored pkspec schemas generated by `pkspec init --dir pkspec`.

Useful commands:

```sh
pkf run spec-check
pkf run spec-lint
pkf run spec-test
pkf run spec-docs
```
